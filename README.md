# 📋 Digital Life Log Planner

**일상을 시각적으로 설계하고 기록하는 모바일 플래너 앱**

Android 모바일 최적화 (430×932px) · 다크/라이트 테마 · 오프라인 지원 (localStorage)

---

## 🚀 시작하기

```bash
# 의존성 설치
pnpm install

# 개발 서버 실행
pnpm dev

# 프로덕션 빌드
pnpm build
pnpm start
```

http://localhost:3000 에서 확인

## ✨ 주요 기능

### 📅 캘린더 헤더 (주간 스트립)
- 요일 라벨(Su~Sa) + 날짜 원형 버튼 7열 그리드
- 오늘(링 표시) / 선택일(채움) 구분
- 일요일(빨강) / 토요일(파랑) 색상 표시
- 좌우 화살표 & 스와이프로 주간 이동
- "주간" 토글 버튼으로 일간↔주간 뷰 전환

### 🕐 일간 뷰 (타임라인)
- 24시간 타임라인 (10분 단위 그리드)
- **계획(plan)** / **실행(execute)** 레이어 시스템
- 터치 드래그로 블록 페인팅 & 지우기
- 블록 캡슐 스타일 표시 (둥근 모서리)
- `createdAt` 기준 블록 정렬
- 기상/취침 인디케이터 (🌅/🌙)
- 체크리스트 FAB 버튼 + 팝업 오버레이

### 📊 주간 뷰
- 7일 컬럼 레이아웃
- 계획/실행 블록 비교 표시
- 활동별 달성률 계산

### 📆 월간 뷰
- 달력 그리드에 일별 활동 요약
- 날짜 클릭 → 일간 뷰로 이동

### ⏱️ 집중 모드 (Focus)
- **카운트다운** (15/25/30/45/60분) + **스톱워치** 모드
- 원형 프로그레스 타이머 (SVG 애니메이션)
- 활동 선택 드롭다운 (하위 활동 포함)
- 완료 시 자동 실행 블록 기록
- 오늘 총 집중 시간 & 세션 완료 횟수 통계

### 🔄 루틴 관리
- 그리드 페인팅 방식으로 루틴 생성
- 일간 루틴 / 주간 루틴 탭
- 터치 드래그로 셀 칠하기 & 지우기
- 루틴 → 타임라인 자동 적용

### 🎨 활동 관리 (설정)
- 활동 추가/편집/삭제
- 하위 활동(서브 카테고리) 지원
- **HSL 컬러 그리드 피커** (14색상 × 6밝기 = 84색 + 그레이스케일)
- 하위 활동: 부모 색상 기반 10단계 변형 팔레트
- 하위 펼침 단계 설정 (0~3단)

### 🛠️ 도구 바 (좌측 사이드바)
- 그리기(Pen) / 지우기(Eraser) 도구 전환
- 계획/실행 레이어 전환
- 활동 팔레트 (클릭으로 선택, 접기/펼치기)
- 하위 활동 자동 표시

### ✅ 체크리스트
- 날짜별 체크리스트 항목 관리
- 완료 체크 & 삭제
- 팝업 오버레이 UI

### 📊 통계/분석
- **주간/월간** 기간 전환 + 이전/다음 네비게이션
- **활동별 실행 시간** 수평 바 차트
- **계획 vs 실행** 달성률 비교 (활동별 %)
- **도넛 차트** 전체 달성률 시각화
- **일별 히트맵** 활동량 밀도 표시
- 총 실행 시간, 일 평균, 세션 통계 카드
- JSON 통계 데이터 내보내기

### 💾 데이터 백업/복원
- 전체 데이터 JSON 백업 내보내기
- JSON 파일에서 백업 불러오기 (자동 복원)

## 📁 프로젝트 구조

```
app/
  layout.tsx          # 루트 레이아웃
  page.tsx            # 메인 페이지
  globals.css         # 전역 스타일 + Tailwind 테마

components/
  life-log-planner.tsx    # 메인 레이아웃 (뷰 라우팅)
  theme-provider.tsx      # 다크/라이트 테마 프로바이더
  planner/
    header.tsx            # 캘린더 주간 스트립 헤더
    bottom-navigation.tsx # 하단 탭 바 (홈/집중/통계/루틴/설정)
    left-sidebar.tsx      # 도구 바 + 활동 팔레트
    timeline.tsx          # 일간 타임라인 (페인팅)
    time-block.tsx        # 블록 렌더링 컴포넌트
    week-view.tsx         # 주간 뷰
    month-view.tsx        # 월간 뷰
    focus-view.tsx        # 집중 모드 (타이머)
    stats-view.tsx        # 통계/분석 (차트, 히트맵)
    routine-view.tsx      # 루틴 관리 (그리드 페인팅)
    settings-view.tsx     # 설정 + 활동 관리
    checklist-panel.tsx   # 체크리스트

lib/
  store.ts            # Zustand 스토어 (persist + immer)
  types.ts            # TypeScript 타입 정의
  utils.ts            # 유틸리티 함수

hooks/
  use-mobile.ts       # 모바일 감지 훅
  use-toast.ts        # 토스트 알림 훅
```

## 📱 뷰 모드

| 모드 | 탭 | 설명 |
|------|-----|------|
| `day` | 홈 | 일간 타임라인 (기본) |
| `week` | 홈 (헤더 토글) | 주간 뷰 |
| `month` | 홈 | 월간 캘린더 |
| `focus` | 집중 | 카운트다운/스톱워치 타이머 |
| `stats` | 통계 | 주간/월간 활동 분석 |
| `routine` | 루틴 | 루틴 생성/관리 |
| `settings` | 설정 | 활동 관리, 테마, 시작 시간 |

## 💾 데이터 저장

- **localStorage** 기반 영구 저장 (Zustand persist)
- 날짜별 블록, 인디케이터, 체크리스트 분리 저장
- Undo/Redo 지원 (블록 히스토리)
- 설정에서 전체 초기화 가능
